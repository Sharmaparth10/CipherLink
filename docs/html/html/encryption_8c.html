Decrypts data using AES-GCM (Authenticated Encryption).

This function decrypts the input data using AES-GCM decryption.

Parameters
ciphertextPointer to the data to decrypt.
ciphertext_lenLength of the ciphertext in bytes.
keyPointer to the decryption key (must be the same as encryption key).
ivPointer to the 12-byte Initialization Vector (must be the same as used in encryption).
plaintextPointer to the buffer where decrypted data will be stored.
plaintext_lenPointer to store the length of the plaintext.
tagPointer to the authentication tag (16 bytes, same as used during encryption).
Returns
SECURE_COMM_SUCCESS on success, or a negative error code on failure.

This function encrypts the input data using AES-GCM encryption.

Parameters
plaintextPointer to the data to encrypt.
plaintext_lenLength of the plaintext in bytes.
keyPointer to the encryption key (16, 24, or 32 bytes for AES-128, AES-192, AES-256).
ivPointer to the 12-byte Initialization Vector (randomly generated).
ciphertextPointer to the buffer where encrypted data will be stored.
ciphertext_lenPointer to store the length of the ciphertext.
tagPointer to store the authentication tag (16 bytes).
Returns
SECURE_COMM_SUCCESS on success, or a negative error code on failure.
